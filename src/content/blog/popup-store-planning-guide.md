## 팝업스토어 기획서, 무엇을 담아야 하는가

팝업스토어는 단순한 "임시 매장"이 아닙니다. IP 세계관을 물리적 공간에 구현하고, 팬들에게 **브랜드 경험**을 제공하는 프로젝트입니다. 따라서 기획서 단계에서 컨셉, 공간, 운영, 예산을 체계적으로 정리해야 합니다.

헤세드코퍼레이션에서 다수의 팝업스토어를 운영한 경험을 바탕으로, 기획서에 반드시 포함해야 할 항목들을 안내합니다.

## 1. 프로젝트 개요

기획서 첫 장에는 프로젝트의 전체 그림을 담습니다.

- **프로젝트명**: IP명 + 팝업 테마 (예: "센과 치히로 × 온천여관 팝업")
- **기간**: 시작일~종료일 (보통 2주~1개월)
- **장소**: 확정 또는 후보 장소 2~3곳
- **목표**: 매출 목표, 방문객 수, SNS 노출 목표
- **타겟 고객**: 핵심 팬층 + 유동 인구 특성

## 2. 컨셉 기획

### 테마 설정

IP의 어떤 요소를 공간에 녹일 것인지 정합니다. 단순히 캐릭터 이미지를 붙이는 것이 아니라, **스토리텔링이 있는 공간**을 만들어야 합니다.

- 작품 속 특정 장소를 재현하는 방식
- 캐릭터의 세계관을 체험할 수 있는 인터랙티브 요소
- 시즌이나 이벤트와 연결한 한정 테마

### 포토존 기획

팝업스토어에서 포토존은 **방문 인증 콘텐츠의 핵심**입니다. SNS 바이럴을 위해 최소 2~3개의 포토존을 기획합니다.

- 대형 캐릭터 조형물 또는 입체 배경
- 조명과 프레이밍이 고려된 촬영 동선
- 해시태그 안내판 배치

## 3. 장소 선정

### 상권별 특성 비교

| 상권 | 유동 인구 | 임대료 수준 | 타겟 연령대 | 적합한 IP |
|------|----------|-----------|-----------|---------|
| 성수동 | 높음 | 높음 | 20~30대 | 트렌디 브랜드, 아트 IP |
| 홍대 | 매우 높음 | 중상 | 10~20대 | 애니/게임/아이돌 IP |
| 여의도 | 중간 | 중간 | 30~40대 | 패밀리, 키즈 IP |
| 잠실 | 높음 | 중상 | 전 연령 | 대중적 캐릭터 IP |

### 공간 체크리스트

- 전용 면적과 층수 (1층 접근성 중요)
- 전력 용량 (조명, POS, 냉난방)
- 화장실, 주차 여부
- 반입/반출 동선 (대형 집기 반입 가능 여부)
- 소방/안전 규정 확인

## 4. 상품 구성(MD Plan)

팝업스토어 전용 상품은 **한정 요소**가 핵심입니다. 일반 유통에서 살 수 없는 상품이 있어야 방문 동기가 생깁니다.

- **팝업 한정 상품**: 전체 SKU의 30% 이상 권장
- **가격대 구성**: 5천원대 소품부터 5만원대 프리미엄까지 분산
- **세트 구성**: 번들 할인으로 객단가 상승 유도
- **구매 특전**: 일정 금액 이상 구매 시 증정품

## 5. 예산 편성

### 주요 비용 항목

| 항목 | 비율 (목안) | 설명 |
|------|-----------|------|
| 장소 임대 | 15~25% | 보증금 + 임대료 |
| 인테리어/집기 | 20~30% | 시공, 포토존, 가구 |
| MD 제작 | 20~25% | 상품 원가 |
| 인력 | 10~15% | 현장 스태프, 보안 |
| 마케팅 | 5~10% | SNS 광고, 인플루언서 |
| 기타 | 5~10% | 보험, 예비비 |

### 손익분기점 산정

예상 일평균 방문객 수와 구매 전환율, 평균 객단가를 기반으로 BEP를 산정합니다.

```
일 매출 = 일 방문객 × 구매 전환율 × 평균 객단가
총 매출 = 일 매출 × 운영 일수
BEP = 총 비용 ÷ (평균 객단가 × 구매 전환율)
```

## 6. 운영 계획

### 일일 운영 타임라인

- **오픈 2시간 전**: 출근, 시재 확인, 진열 정리
- **오픈**: POS 가동, 입장 관리 시작
- **피크 타임 (12~15시)**: 인력 집중 배치
- **마감 1시간 전**: 재고 중간 집계
- **마감 후**: 정산, 시재 마감, 청소, 보안 점검

### 인력 구성

- 매장 매니저 1명 (총괄)
- 판매 스태프 2~4명 (면적/예상 인원에 따라)
- 입장 관리 1명 (대기열이 예상되는 경우)

## 마무리

좋은 팝업스토어 기획서는 "이 공간에서 고객이 어떤 경험을 하게 될 것인가"를 구체적으로 보여줍니다. 컨셉, 공간, 상품, 예산, 운영이 하나의 흐름으로 연결될 때, 기획서의 설득력이 높아집니다.
